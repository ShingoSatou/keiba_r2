# History

このディレクトリは、旧 repo の raw log を再配置する場所ではありません。  
研究を続けるときに必要な「要約された判断」と「archive を引くときの読み方」だけを残します。

## 何が入っているか
- `active-asset-root.md`
  - この workspace で現在使っている `V3_ASSET_ROOT` の場所
- `decision-log.md`
  - この repo の前提にした恒久判断
- `migration-ledger.md`
  - 何を新 repo に持ち込み、何を持ち込まなかったか
- `milestones.md`
  - 研究運用を変えた主要な節目
- `open-hypotheses.md`
  - まだ閉じていない仮説
- `operational-lessons.md`
  - 再発防止のための運用ルール
- `archive-map.md`
  - raw provenance が必要なときに、旧 archive で何を探すか

## このディレクトリの使い方
- 通常の実装や研究では、まず `docs/specs/` を見ます。
- 歴史的判断や過去の失敗理由が必要なときだけ `docs/history/` を見ます。
- archive から何かを持ち込むときは、raw doc を複製するのではなく、このディレクトリに要点を要約して追加します。
